# Project Refactor: 游戏策划总案 (v3.0)

**项目名称：** Project Refactor (重构人生)
**版本号：** v3.0 (The Soul & Fog Update)
**核心体验：** 在认知迷雾中，通过重构灵魂属性，探索阶层突围的可能性。
**一句话：** 这不是模拟经营，这是“幸存者偏差”模拟器。

---

## 1. 序章：灵魂捏脸 (Soul Sculpting)

游戏开始时，玩家不对外貌进行调整，而是对“灵魂”的初始参数进行分配。这决定了玩家看待世界的方式（即迷雾的形态）。

### 1.1 四大核心维数 (The 4 Dimensions)
初始点数：20点（可微调）。

| 属性 (Stat) | 变量名 | 作用 (Function) | 现实映射 |
| :--- | :--- | :--- | :--- |
| **安全感** | `Security` | **抗压护盾**。决定高风险路径（创业/全职考研）的可见性。 | 家庭兜底能力、存款 |
| **信息熵** | `Entropy` | **视野广度**。决定稀有路径（留学/特殊红利）的可见性。 | 见世面、信息渠道 |
| **自尊值** | `Pride` | **精神杠杆**。决定底层/求助路径的可见性。过高易产生“孔乙己”Debuff。 | 面子、心气 |
| **执行力** | `Focus` | **长线输出**。决定长周期项目（需要连续投入N月）的成功率。 | 专注力、定力 |

### 1.2 出身模版 (Origins)

| 出身 (Class) | 属性倾向 | 初始迷雾状态 | 特殊被动 (Buff/Curse) |
| :--- | :--- | :--- | :--- |
| **小镇做题家** | Focus ++<br>Entropy -- | 仅【考研】【工作】可见。<br>【留学】【创业】隐形。 | **【题海】**：考试类事件成功率 +30%。<br>**【井底】**：探索迷雾消耗 AP +50%。 |

| **野蛮生长者** | Security ++<br>Pride -- | 【搞钱】路径全开。<br>【学术】路径隐形。 | **【厚脸皮】**：向 NPC 借钱/求助成功率极高。<br>**【短视】**：长线项目每回合 Focus 衰减加快。 |
| **一无所有者** | 平均 | 标准迷雾。 | 无。 |

## 2. 核心系统：认知迷雾 (Cognitive Fog)

游戏的核心玩法不是“走地图”，而是“开地图”。人生的可能性被一层黑雾笼罩。

### 2.1 迷雾状态机
对于每一个人生节点（如“申请常春藤”、“去深圳创业”、“考公”），有三种状态：

1.  **隐形 (Invisible):**
    * 表现：UI 上根本没有这个按钮。
    * 含义：认知盲区。玩家不知道“还能这样活”。
2.  **模糊 (Blurred):**
    * 表现：显示为灰色 `???` 节点，或是“遥远的彼岸”。
    * 含义：知道有这条路，但不知道怎么走，或者觉得自己不配。
    * *转化条件：* 属性达标（如 `Entropy > 60`）或 关键事件触发。
3.  **清晰 (Clear):**
    * 表现：点亮，显示具体 AP 消耗和成功率。
    * 含义：路径明确，只差执行。

### 2.2 迷雾回潮 (Fog Regression)
迷雾是动态的。
* **打击回潮：** 当 `Security` 或 `Pride` 遭受重创（如被裁员、投资失败），原本清晰的**高风险/高门槛节点**会重新变回【模糊】甚至【隐形】。
* **绝处逢生：** 陷入绝境时，原本不可见的【底层生存】节点（如三和大神）会突然变得【清晰】。

## 3. 探索系统：碎片与引导 (Exploration)

地图（大学城）是信息的容器。

### 3.1 碎片化叙事 (Fragments)
玩家通过点击场景、撞击建筑、与 NPC 对话来收集碎片。
* **实物碎片：** `雅思传单`、`选调生公告`、`大厂工牌`。 -> **直接增加对应迷雾节点的“发现进度”**。
* **对话碎片：** 与 Xuan 闲聊，得知他要出国。 -> **解锁【留学】节点的线索**。

### 3.2 建筑功能 (Buildings)
* **图书馆 (Library):** 消耗 AP，提升 `Entropy` (开迷雾) 或 `Settlement` (攒大招)。
* **宿舍 (Dorm):** 恢复 AP。如果 `Anxiety` 过高，恢复效率降低（睡不着）。
* **便利店 (Shop):** 购买道具（咖啡/红牛/褪黑素）来调整当月状态。

## 4. 循环机制：月度规划 (Monthly Planning)

**UI:** 每月 1 号弹出的“手账本”。
**资源:** 3 个【专注槽 (Focus Slot)】。

### 4.1 策略选择
玩家需将 3 个槽分配给不同方向：
* **[搞钱]:** 本月打工收益 $\uparrow$，迷雾探索率 $\downarrow$。
* **[死磕]:** 本月学业/项目进度 $\uparrow$，焦虑增长 $\uparrow$。
* **[开眼]:** 本月迷雾探索概率 $\uparrow$ (易发现新路)，金钱收益 0。
* **[社交]:** NPC 好感度 $\uparrow$，获得额外信息。

### 4.2 诛心反馈 (The Inner Voice)
在月度结算界面，根据玩家状态生成红字文本。
* *场景：* 玩家没钱且焦虑高。
* *文本：* “这杯咖啡 35 块，是你两天的食堂伙食费。你伸出的手又缩了回去。”

## 5. Boss 战：阶段性大考 (Milestones)

**定义：** 人生关键节点的具象化战斗。
**Boss 列表：** 【大三实习】、【考研上岸】、【毕业答辩】、【家庭变故】。

### 5.1 战斗逻辑 (Card Battle)
不是动作战斗，而是基于**沉淀值 (Settlement)** 和 **特殊经历 (Experience)** 的卡牌判定。

* **Boss 阶段 1 (笔试/筛选):**
    * Boss 技能：`枯燥轰炸` (造成 AP 损伤)。
    * 判定：`Settlement > 阈值` 则无伤过关，否则掉血。
* **Boss 阶段 2 (面试/刁难):**
    * Boss 技能：`压力测试` (造成自尊伤害)。
    * 玩家出牌：打出【大厂实习证明】(免疫伤害并反击) 或 【雅思 7.5】(造成暴击)。
* **结局:**
    * **Win:** 获得【名校/名企】Buff，下一章初始资源提升。
    * **Lose:** 进入兜底路线（二战/普通工作），迷雾重组。

## 6. 基础数值公式 (Formulas)

### 6.1 焦虑增长 (Anxiety Delta)
$$\Delta A = (BaseStress \times P_{sens}) - (Settlement \times 0.1)$$

### 6.2 胆量 (Boldness)
*决定是否敢于选择“高自尊”选项（如拒绝不合理要求）。*
$$Boldness = (P_{fin} \times 0.4) + (P_{pride} \times 0.6)$$

### 6.3 迷雾可见分 (Visibility Score)
*决定某个节点是否可见。*
$$V_{score} = (Entropy \times w_1) + (Security \times w_2) + \sum(Fragments)$$
