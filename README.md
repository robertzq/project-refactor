# 📖 关于项目 (About)
Project Refactor 是一款基于 Godot 引擎 开发的 2.5D 生活模拟与策略游戏。

不同于传统的“养成游戏”堆砌数值，本项目旨在探讨 “幸存者偏差” 与 “认知局限”。在游戏中，你扮演的不仅是一个刚毕业的大学生，更是一个试图修补自我、对抗熵增的“重构者”。

你需要在一个充满噪音的城市中，平衡金钱（生存底气）、自尊（精神杠杆）与焦虑（系统负载），在迷雾中寻找通往未来的路径。

## ✨ 核心特性 (Key Features)
### 1. 🌫️ 认知迷雾系统 (The Cognitive Fog)
“你无法选择你看不见的路。” 游戏取消了上帝视角。你的人生技能树被厚重的迷雾笼罩。

眼界 (Entropy) 决定了你的视野半径。只有通过沉淀与学习提升眼界，那些原本“不存在”的机遇才会显现。

在此之前，你只能在有限的选项中“随波逐流”。

### 2. 🧬 差异化出身 (Origin Archetypes)
没有捏脸，只有一张通往大学城的 “单程车票”。 开局选择决定了你的底层代码架构：

小镇做题家 (The Striver)： 极高的执行效率，极低的容错率。不仅要对抗贫穷，还要对抗敏感。

落魄书香 (The Scholar)： 高傲的自尊是你的护甲，也是你的软肋。

野蛮生长 (The Hustler)： 低自尊换来的是生存的韧性，只要能活下去，什么都能做。

温室花朵 (The Heir)： 自带“退路”机制，但容易陷入安逸的死循环。

### 3. ⚖️ 动态压力博弈 (Anxiety Engine)
焦虑不是一个简单的血条，而是一套复杂的动态平衡系统：

金钱护盾： 存款能直接抵消生存类焦虑。

自尊棘刺： 过高的自尊会导致你在遭遇挫折时受到成倍的精神伤害（Ego Damage）。

避难所机制： 当压力过载时，工作、图书馆甚至一次彻底的崩溃，都可能成为你的临时避难所。

### 4. 🕰️ 耕耘与收获
复刻了类似《星露谷物语》的耕耘体验，但你耕种的不是农作物，而是 “项目代码” 与 “人生课题”。

你需要管理你的 精力 (CPU)，在图书馆的角落里，一行行敲出你的未来。

Design Philosophy: Object-Oriented Programming applied to Sociology.
